# 🚀 Quick Start - Instant Deployment!

## Option 1: Standalone HTML (NO INSTALLATION REQUIRED!) ⚡

**The standalone-visualizer.html file runs directly in your browser!**

### Steps:
1. **Download** the `standalone-visualizer.html` file
2. **Double-click** it to open in your browser
3. **Done!** The visualizer is now running!

✅ No npm install needed
✅ No dependencies to download
✅ Works offline after first load
✅ All libraries loaded from CDN

This is the **FASTEST** way to run the visualizer!

---

## Option 2: Full React Development Setup

If you want the full development environment:

### Prerequisites:
- Node.js 16+ installed
- npm or yarn

### Steps:
```bash
# 1. Navigate to project folder
cd sorting-visualizer

# 2. Install dependencies
npm install

# 3. Start dev server
npm run dev

# 4. Open browser at http://localhost:3000
```

---

## 🎯 Which Option Should You Use?

**Use Standalone HTML if:**
- You just want to see it working NOW
- You don't have Node.js installed
- You want to share with someone quickly
- You want offline capability

**Use Full Setup if:**
- You want to modify the code
- You want hot-reload during development
- You want to build for production
- You want the best developer experience

## 💡 Tips:

- **Speed**: Try 80%+ for fast animations, 20% for detailed learning
- **Array Size**: 30-50 elements is optimal for watching algorithm behavior
- **Sound**: Turn on sound effects for extra feedback
- **Custom Arrays**: Test specific cases like [5,4,3,2,1] or [1,2,3,4,5]

---

## 🐛 Troubleshooting:

**Standalone HTML not working?**
- Make sure you have internet connection (first load only)
- Try a different browser (Chrome/Firefox/Safari/Edge)
- Check browser console for errors (F12)

**Full setup not installing?**
- Make sure Node.js 16+ is installed: `node --version`
- Delete node_modules and package-lock.json, try again
- Use `npm ci` instead of `npm install`
